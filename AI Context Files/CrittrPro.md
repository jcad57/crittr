## This document outlines the features available to pro members as well as non-pro members

# FREE

1. User account
2. One pet
3. Unlimited activity logs
4. Add up to one meal, treat, medication and vaccination

# PRO

1. CrittrAI
2. Unlimited pets
3. Co-care feature
4. Upload pet records
5. Store unlimited meals/treats/medications
6. Notifications and Reminders
