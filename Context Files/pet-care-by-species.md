## Pet Care by Type

This document outlines the different requirements for each type of pet we currently support in this app. It will show how dashboards will reflect differently based on the active pet type. For dogs you might see daily progress for feeding, exercise and treats while birds might show feeding, free-fly time, cage cleaning, etc.

The purpose of this app is to help manage all types of different pets by provided individualized management options for all different types of pets. Below you will see what each dashboard might look like based on the pet type.

# Dogs

Daily progress bar:

1. Exercise: All forms of exercise from walks to hikes to dog park visits.
2. Meals: Breakfast, dinner, desert, etc.
3. Treats: Track treats given throughout the day based on the set threshold.
4. Meds: Track medications required for the day.

Activities:
All the above plus the user can log vet visits.
